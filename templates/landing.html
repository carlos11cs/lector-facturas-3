FAQ

Resolvemos las dudas habituales sobre software para gestorías.

¿Está pensado para gestorías o para pymes?

Es un software para gestorías. Está diseñado para trabajar con múltiples empresas, equipos y reporting trimestral.

¿Se pueden gestionar varias empresas?

Sí. Puedes centralizar todas las empresas cliente y cambiar de una a otra con filtros claros.

¿Puede trabajar más de un miembro del equipo?

Sí. Puedes añadir usuarios staff y asignar empresas a cada persona para una operativa ordenada.

¿Qué pasa si el OCR se equivoca?

Ledged marca incoherencias y permite corregir manualmente. La automatización nunca bloquea el trabajo.

¿Cómo funciona el reporting trimestral?

Genera informes de IVA, P&G y balance, listos para enviar por email con un clic.
